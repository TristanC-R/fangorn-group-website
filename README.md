# Fangorn Group Website (Concept)

This repo is a runnable dev project created from `fangorn_website_concept.jsx`.

## Quick start

```bash
npm install
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:5173`).

## Where to edit

- Main page component: `src/components/FangornWebsite.jsx`
- App wrapper: `src/App.jsx`
